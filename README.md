# JobScope: AI-Powered Job Intelligence Platform

JobScope is a data/backend/AI project that collects, processes, and stores job posting data, then provides skill analytics, similar job recommendations, and RAG-based question answering.

## Features
- Ingest job posting data from CSV into PostgreSQL
- Provide job listing API
- Analyze frequently required skills
- Recommend similar jobs based on semantic embeddings
- Support RAG-based Q&A with source chunks

## Tech Stack
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Pandas
- sentence-transformers
- FAISS
- Docker
